## Features

- Shorten long URLs to clean, memorable links
- Track number of clicks for each URL
- View recently created URLs
- Copy short URLs to clipboard
- Delete URLs from history
- Secure authentication with **JWT access and refresh tokens**
- Logout functionality with token blacklist support

---

## Tech Stack

- Backend: Django , Django REST Framework, Simple JWT
- Frontend: HTML, CSS,  JavaScript
- Database: SQLite (default)
- Authentication: JWT (access + refresh tokens)
- Local storage used for recent URLs history(cookies)
